Project Name:
Weather Forecast Web Application
----
Project Description
This application enables users to:
Enter a city name through a web interface.
Fetch real-time weather data from the OpenWeather API.
Display current weather information such as:
Temperature
Humidity
Wind Speed
Weather Condition
Pressure
Visibility
Generate and display a multi-day weather forecast.
Handle invalid city names and API-related errors gracefully.
------
Main Features
✅ Current Weather Monitoring
✅ 5-Day Weather Forecast
✅ City Search Functionality
✅ Weather Icons Display
✅ Error Handling for Invalid Locations
✅ Responsive Django Web Application
✅ API Integration with External Weather Service
